# 智能会议记录生成器

## 项目简介
基于Python Flask开发的Web应用，通过大模型深度解析用户输入的会议内容，自动提取关键信息并生成规范的Word格式会议记录。

## 功能特点
- 📝 智能解析会议文本内容
- 🤖 支持本地Ollama大模型集成
- 📄 自动生成标准Word文档
- 🎨 友好的Web界面
- 📱 响应式设计

## 安装和运行

### 环境要求
- Python 3.7+
- Ollama (可选，用于大模型功能)
